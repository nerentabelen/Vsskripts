    !4 тема: Преобразование типов данных 
JavaScript автоматически выполняет преобразования типов в различных контекстах, но также он позволяет явно выполнить такие преобразования:
a = "123", (string)
b = 5,
newA = Number(a) (превращает a - в число)
newB = String(b) (превращает b - в строку)
console.log(newA+b)
console.log(a+newB)

c = 3,14,
newC = parseInt(c); (превращает число с десятыми в целое)
console.log(newC)

Number - пытается преобразовать переменную в числовой тип данных.
String - пытается преобразовать переменную в строчный тип данных.
parseInt - пытается преобразовать числовой тип данных и убирает числа после запятой.
parsefloat - пытается преобразовать числовой тип данных и не убирает числа после запятой.

тема 6 операторы сравнения

1 < условие верно если первый оперант меньше второго (true)
2 > условие верное если первый оперант больше второго
3 <= условие верное если первый оперант меньше или равен второму
4 >= условие верное если первый оперант больше или равен второму
5 == условие верное если операнты равны
6 === условие верное если оба операнта равны и имеют одинаковый тип данных 

тема 7 условные операторы (if,else)


условные операторы позволяют выполнять различные действия в зависимости от выполнения определенных условий.
основные операторы в js это if(), else() if{}, else{}.
инструкция if() вычесляет выражение в круулых скобках и преобразует результат к логическому типу 
let x = + prompt("введите число");

console.log("икс больше 5");
}
else if(x===5){
console.log("икс равен 5")
}
else{
console.log("икс меньше 5")
}
